# Query the Fast Radio Burst Catalogue in Python
**David Gardenier** | ASTRON

## Installation

```sh
pip3 install frbcat
```

## Usage
Get a Pandas DataFrame using
```python
from frbcat import Frbcat
df = Frbcat().df
```
